# Distance Measuring Using HC-SR04

## Description
A part of my graduation project that measures distance using raspberry pi and ultrasonic sensor and takes actions according to the distance

## Hardware Components:
* Raspberry Pi 3 Model B
* HC-SR04 sensor
* Buzzer
* LEDs
* Jumper wires
